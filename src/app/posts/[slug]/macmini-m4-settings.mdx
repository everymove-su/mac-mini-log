맥미니 M4를 처음 켰을 때, 기본 설정 그대로 쓰는 분들이 많아요.

하지만 몇 가지만 바꿔도 체감 성능과 편의성이 확 달라집니다.

개봉 후 바로 해야 할 핵심 설정만 골랐어요.

---

## 1. 🔋 에너지 절약 설정 — 성능 모드로 바꾸자

기본 설정은 에너지 효율 우선이라 성능이 살짝 제한될 수 있어요.

무거운 작업을 자주 한다면 고성능 모드를 켜두는 게 이득입니다.

```
시스템 설정 › 에너지 절약 › 고성능 모드 활성화
```

> 💡 맥미니는 배터리가 없으니 고성능 모드를 켜도 부담 없어요. 일반 사용자라면 자동 모드도 충분합니다.

---

## 2. 🖥️ 디스플레이 해상도 — 기본값이 최선이 아닐 수 있다

모니터를 연결하면 macOS가 자동으로 해상도를 잡아줍니다.

하지만 텍스트가 너무 작거나 화면이 흐릿하다면 직접 맞춰주세요.

```
시스템 설정 › 디스플레이 › 해상도: 조절된 항목 선택
```

4K 모니터라면 **"더 많은 공간"** 옵션이 생산성에 유리하고, 눈이 편하려면 **"기본"** 또는 한 단계 낮춘 해상도를 권장합니다.

> ⚠️ 외부 모니터 주사율(60Hz/120Hz)도 꼭 확인하세요. 지원 모니터라면 주사율을 올려야 마우스 커서가 훨씬 부드러워집니다.

---

## 3. 🌙 Night Shift — 눈 건강은 기본 중의 기본

저녁부터 자동으로 화면 색온도를 따뜻하게 바꿔주는 기능이에요.

처음엔 어색할 수 있지만, 며칠 쓰면 없으면 오히려 불편합니다.

```
시스템 설정 › 디스플레이 › Night Shift › 일몰부터 일출까지
```

> 💡 색보정 작업을 하는 분이라면 Night Shift는 끄고 작업하세요. 색상이 왜곡되어 보일 수 있어요.

---

## 4. 🧲 Dock 자동 숨기기 — 화면을 더 넓게 쓰자

Dock이 항상 보이면 생각보다 화면을 많이 차지합니다.

자동 숨기기를 켜면 사용할 때만 나타나서 작업 공간이 확 넓어져요.

```
시스템 설정 › 데스크탑 및 Dock › Dock 자동으로 가리기 및 표시 켜기
```

크기와 확대 효과도 취향에 맞게 조절해보세요.

Dock 위치를 왼쪽이나 오른쪽으로 옮기면 세로 공간을 더 살릴 수 있어요.

---

## 5. 🗂️ 스테이지 매니저 — 창 정리의 신세계

여러 창을 동시에 열고 작업하는 분께 강력 추천하는 기능이에요.

앱을 그룹으로 묶어서 깔끔하게 전환할 수 있습니다.

```
제어 센터 › 스테이지 매니저 클릭
```

또는 설정에서 직접 켤 수도 있어요.

```
시스템 설정 › 데스크탑 및 Dock › 스테이지 매니저
```

> 💡 처음엔 어색할 수 있어요. 3일만 써보면 적응됩니다. 안 맞으면 그냥 끄면 그만이에요!

---

## 6. ⚡ 핫 코너 — 마우스 한 번에 원하는 동작

화면 모서리에 마우스를 갖다 대면 특정 기능이 실행되는 설정입니다.

한 번 익숙해지면 없이는 못 살아요.

```
시스템 설정 › 데스크탑 및 Dock › 핫 코너
```

**추천 조합**

- 왼쪽 하단 → 화면 잠금
- 오른쪽 하단 → 데스크탑 표시
- 오른쪽 상단 → Mission Control

---

## 7. 🖱️ 마우스 속도 조절

기본 마우스 속도가 느리다고 느끼는 분들이 많아요.

특히 고해상도 모니터를 쓴다면 반드시 조절이 필요합니다.

```
시스템 설정 › 마우스 › 추적 속도 조절
```

macOS 기본 마우스 가속도가 불편하다면 **LinearMouse** 같은 무료 앱도 추천합니다.

---

## 8. ☁️ iCloud 동기화 — 필요한 것만 켜자

처음 설정하면 모든 항목이 켜져 있는 경우가 많아요.

사용하지 않는 항목까지 동기화되면 저장공간도 줄고 속도도 느려질 수 있어요.

```
시스템 설정 › Apple 계정 › iCloud › 앱별 동기화 항목 선택
```

> 💡 사진, 메모, 연락처 정도만 켜도 충분한 경우가 많아요. 나중에 필요하면 항목별로 추가하면 됩니다.

---

## 9. 🔐 화면 잠금 시간 설정

자리를 비울 때 자동으로 화면이 잠기는 시간을 설정해두세요.

보안의 기본 중 기본이에요.

```
시스템 설정 › 잠금 화면 › 잠자기 또는 화면 보호기 시작 후 암호 요구: 즉시
```

Apple Watch를 갖고 있다면 **Apple Watch로 Mac 잠금 해제** 기능도 꼭 켜두세요.

```
시스템 설정 › Touch ID 및 암호 › Apple Watch로 잠금 해제
```

---

## 10. 🔄 소프트웨어 업데이트 자동화

보안 패치는 최대한 빠르게 적용하는 게 좋아요.

자동 업데이트를 켜두면 신경 쓸 일이 줄어듭니다.

```
시스템 설정 › 일반 › 소프트웨어 업데이트 › 자동 업데이트 켜기
```

> ⚠️ 큰 업데이트(macOS 버전 업)는 바로 올리기보다 한 달 정도 기다렸다가 올리는 걸 추천해요. 초기 버그를 피할 수 있거든요.

---

처음 설정이 가장 중요합니다. 10가지 다 하는 데 10분도 안 걸려요. 🍎
